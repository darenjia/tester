**修改的文件:** `utils/logger.py`

**问题:** `get_logger()` 函数不接受参数，但多处代码传了名称参数

**修复:** 修改 `get_logger()` 函数接受可选的 `name` 参数
```python
def get_logger(name: str = None) -> logging.Logger:
    if name:
        return logging.getLogger(name)
    return logger_manager.get_logger()
```

**修改的文件:** `utils/metrics.py`
- 修复 `get_logger("metrics")` 调用
