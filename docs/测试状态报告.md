## 修复的问题

### 1. is_connected属性修复

**问题**: `is_connected` 属性只检查 `AdapterStatus.CONNECTED` 状态，但 `RUNNING` 状态也应该被认为是已连接。

**修复**: 修改 `base_adapter.py` 中的 `is_connected` 属性：

```python
@property
def is_connected(self) -> bool:
    """检查是否已连接（包括运行中状态）"""
    return self.status in [AdapterStatus.CONNECTED, AdapterStatus.RUNNING]
```

**影响**: 修复了基础适配器测试中的 `test_start_test_changes_status` 测试失败。
