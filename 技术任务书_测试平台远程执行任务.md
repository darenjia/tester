## 1. 项目背景

### 1.1 项目概述

本项目旨在构建一个**测试任务调度平台**，实现从Excel数据导入、平台处理、API传输到设备指令下发的完整自动化流程。系统采用分层架构设计，通过下发测试任务的方式，调用远程设备上的**CANoe**、**TSMaster**或**TTworkbench**测试软件执行自动化测试，并将测试结果上报至平台。

### 1.2 业务场景

- 测试人员通过Web界面创建和下发测试任务，或通过Excel模板批量导入测试配置
- 平台将任务分发到部署有测试软件的远程执行设备
- 远程设备执行测试（CANoe/TSMaster/TTworkbench自动化测试）
- 测试结果自动上报至平台进行展示和分析

### 1.3 测试系统机柜接入

本系统需接入以下三类测试系统机柜：

| 测试系统 | 执行软件 | 接入方案 | 状态 |

| 网络自动化测试系统机柜 | CANoe 10 | C#脚本调用，开发API接口对接 | 待开发 |
| 便携式网络自动化测试系统机柜 | CANoe 17 + TAMS | 基于CANoe COM接口二次开发 | 待开发 |

## 2. 目标与范围

### 2.1 总体目标

实现测试平台"**下发任务**"功能：
> 通过 **Java 调用 Python 封装的接口**，在远程设备上驱动 CANoe/TSMaster/TTworkbench 执行测试任务，并获取执行结果。

### 2.2 具体目标

1. **设备管理**：远程执行设备的注册、状态监控、心跳检测
2. **任务分发**：Java服务端将任务下发到指定设备的Python执行器
3. **执行控制**：Python端接收任务，调用CANoe/TSMaster/TTworkbench API执行测试
4. **结果上报**：测试完成后，结果自动回传至Java服务端
5. **实时监控**：测试执行过程中的日志和状态实时推送
6. **Excel导入**：支持Excel模板批量导入测试配置参数

### 2.3 范围界定

**包含范围**：
- Java服务端与Python执行器之间的通信协议设计
- Python端CANoe/TSMaster/TTworkbench接口封装
- 任务执行状态机管理
- 测试结果数据采集与上报
- Excel数据导入与字段解析
- 三类测试系统机柜的接入方案设计

**不包含范围**：
- CANoe/TSMaster/TTworkbench测试用例本身的开发
- 硬件设备的物理连接配置
- TDM 2.0接口的进一步扩展
- TTworkbench TSN测试的参数配置接口（已确认不支持）

**接入方案**: C#脚本调用，开发API接口对接

**实现步骤**:
1. 获取原测试软件C#源码
2. 更改程序增加与网络测试管理平台对接的API接口
3. 增加用例与脚本匹配策略程序
4. 通过Python脚本调用C#程序接口

**数据流向**:
```
测试平台 → Python执行器 → C#程序 → CANoe 10 → 测试结果
```

#### 4.2.2 便携式网络自动化测试系统机柜接入

**测试软件**: CANoe 17 + TAMS

**接入方案**: 基于CANoe COM接口二次开发

**方案选择**:
- 方案一：基于现有TAMS软件接口开发（待厂家确认TAMS是否开放API）
- 方案二：基于CANoe COM接口进行二次开发（推荐）

**实现流程**:
1. 测试机柜电脑安装Python客户端软件
2. 客户端与测试管理平台通讯获得具体测试任务
3. 根据任务控制执行软件选择特定脚本进行测试执行
4. 通过调用API接口获得测试结果
5. 将测试结果上传给测试管理平台

**数据流向**:
```
测试平台 → Python执行器 → CANoe COM接口 → CANoe 17 → 测试结果
```

#### 4.2.3 车载以太网测试系统机柜接入

**测试软件**: TTworkbench + CANoe

**接入方案**: TTman命令行调用clf文件

**核心组件**:
- **TTman**: 测试执行管理器，负责加载编译好的测试套件、调度测试用例执行、生成测试报告
- **TTthree**: 编译器，将TTCN-3源代码编译成可执行代码
- **TTsuite**: 测试套装，包括OPEN Alliance SIG一致性测试（TC8）、汽车AVB一致性测试、AUTOSAR一致性测试等

**测试流程**:
1. 环境准备
   - 连接Test System与DUT
   - 加载对应的TTsuite
   - 配置TTsuite参数（IP地址、MAC地址等）
   - 启动UT

2. 执行测试
   - 运行测试脚本
   - 获得测试数据和测试报告

**TTman命令调用示例**:
```bash
TTman.bat --data <workspace_Path> --log C:\Spirent\Logs -r pdf --report-dir C:\Spirent\Logs <CLF_Path>\TestCase.clf
```

**参数说明**:
- `--data`: 指定workspace路径
- `--log`: 指定tlz log导出路径
- `-r pdf`: 指定报告格式为PDF
- `--report-dir`: 指定测试report导出路径
- `<CLF_Path>`: 要调用的clf文件路径

**数据流向**:
```
测试平台 → Python执行器 → TTman命令 → TTworkbench → 测试结果
```

**注意事项**:
- TSN测试时需要有信息交互和参数输入确认，思博伦软件接口实现困难
- 参数配置接口已确认不支持
- 需要预先在TTworkbench客户端创建可执行的clf文件
- 每个clf文件对应一个测试用例，通过设置Runs参数控制执行次数

```python
"""
CANoe控制器 - 基于COM接口
"""
import win32com.client
import time
import logging
from typing import Optional, Dict, Any

class CANoeController:
    """CANoe测试软件控制器"""
    
    def __init__(self):
        self.app = None
        self.measurement = None
        self.logger = logging.getLogger(__name__)
        
    def connect(self) -> bool:
        """连接CANoe应用"""
        try:
            self.app = win32com.client.Dispatch("CANoe.Application")
            self.measurement = self.app.Measurement
            self.logger.info("CANoe连接成功")
            return True
        except Exception as e:
            self.logger.error(f"CANoe连接失败: {e}")
            return False
    
    def open_configuration(self, cfg_path: str) -> bool:
        """加载CANoe配置文件"""
        try:
            self.app.Open(cfg_path)
            self.logger.info(f"配置文件加载成功: {cfg_path}")
            return True
        except Exception as e:
            self.logger.error(f"配置文件加载失败: {e}")
            return False
    
    def start_measurement(self) -> bool:
        """启动测量"""
        try:
            if not self.measurement.Running:
                self.measurement.Start()
                # 等待启动完成
                timeout = 30
                while not self.measurement.Running and timeout > 0:
                    time.sleep(0.5)
                    timeout -= 0.5
                if self.measurement.Running:
                    self.logger.info("测量已启动")
                    return True
            return False
        except Exception as e:
            self.logger.error(f"启动测量失败: {e}")
            return False
    
    def stop_measurement(self) -> bool:
        """停止测量"""
        try:
            if self.measurement.Running:
                self.measurement.Stop()
                self.logger.info("测量已停止")
            return True
        except Exception as e:
            self.logger.error(f"停止测量失败: {e}")
            return False
    
    def read_signal(self, channel: int, message: str, signal: str) -> Optional[float]:
        """读取信号值"""
        try:
            bus = self.app.BusSystems(channel)
            value = bus.Signals.Item(signal).Value
            return float(value)
        except Exception as e:
            self.logger.error(f"读取信号失败: {e}")
            return None
    
    def write_signal(self, channel: int, message: str, signal: str, value: float) -> bool:
        """写入信号值"""
        try:
            bus = self.app.BusSystems(channel)
            bus.Signals.Item(signal).Value = value
            return True
        except Exception as e:
            self.logger.error(f"写入信号失败: {e}")
            return False
    
    def run_test_module(self, test_name: str) -> Dict[str, Any]:
        """执行测试模块"""
        try:
            test_config = self.app.TestConfiguration
            test_config.ExecuteTest(test_name)
            
            # 获取测试结果
            result = {
                "test_name": test_name,
                "verdict": test_config.Verdict,
                "start_time": test_config.StartTime,
                "end_time": test_config.EndTime
            }
            return result
        except Exception as e:
            self.logger.error(f"执行测试失败: {e}")
            return {"error": str(e)}
    
    def disconnect(self):
        """断开连接"""
        try:
            if self.measurement and self.measurement.Running:
                self.measurement.Stop()
            self.app = None
            self.logger.info("CANoe连接已断开")
        except:
            pass
```

"""
TSMaster控制器 - 基于原生Python API
"""
import logging
from typing import Optional, Dict, Any, Callable

try:
    from TSMaster import *
    TSMASTER_AVAILABLE = True
except ImportError:
    TSMASTER_AVAILABLE = False

class TSMasterController:
    """TSMaster测试软件控制器"""
    
    def __init__(self):
        self.ts = None
        self.logger = logging.getLogger(__name__)
        self.callbacks = {}

            self.logger.info("TSMaster连接成功")

        """设置信号值"""
        try:
            self.ts.set_signal_value(signal_name, value)
            return True
        except Exception as e:
            self.logger.error(f"设置信号失败: {e}")
            return False
    
    def send_message(self, channel: int, msg_id: int, data: list) -> bool:
        """发送CAN报文"""
        try:
            self.ts.transmit_can_msg(channel, msg_id, data)
            return True
        except Exception as e:
            self.logger.error(f"发送报文失败: {e}")
            return False
    
    def register_rx_callback(self, callback: Callable):
        """注册报文接收回调"""
        try:
            self.ts.register_rx_callback(callback)
        except Exception as e:
            self.logger.error(f"注册回调失败: {e}")
    
    def run_test_sequence(self, sequence_name: str) -> Dict[str, Any]:
        """执行测试序列"""
        try:
            # TSMaster测试执行
            result = self.ts.test_execute(sequence_name)
            return {
                "sequence_name": sequence_name,
                "result": result,
                "status": "completed"
            }
        except Exception as e:
            self.logger.error(f"执行测试序列失败: {e}")
            return {"error": str(e)}
    
    def disconnect(self):
        """断开连接"""
        try:
            if self.ts:
                self.ts.stop_bus()
                self.ts.disconnect()
            self.logger.info("TSMaster连接已断开")
        except:
            pass

"""

import subprocess
import os
import logging
from typing import Optional, Dict, Any

class TTworkbenchController:
    """TTworkbench测试软件控制器"""
    
    def __init__(self):
        self.ttman_path = None
        self.workspace_path = None
        self.log_path = None
        self.report_path = None
        self.logger = logging.getLogger(__name__)
        
    def configure(self, ttman_path: str, workspace_path: str, 
                 log_path: str, report_path: str) -> bool:
        """配置TTworkbench路径"""
        try:
            self.ttman_path = ttman_path
            self.workspace_path = workspace_path
            self.log_path = log_path
            self.report_path = report_path
            
            # 验证路径存在
            if not os.path.exists(self.ttman_path):
                raise Exception(f"TTman路径不存在: {self.ttman_path}")
            if not os.path.exists(self.workspace_path):
                raise Exception(f"Workspace路径不存在: {self.workspace_path}")
                
            self.logger.info("TTworkbench配置成功")
            return True
        except Exception as e:
            self.logger.error(f"TTworkbench配置失败: {e}")
            return False
    
    def execute_test(self, clf_file: str) -> Dict[str, Any]:
        """执行测试用例
        
        Args:
            clf_file: clf文件路径
            
        Returns:
            执行结果字典
        """
        try:
            # 构建命令
            cmd = [
                self.ttman_path,
                '--data', self.workspace_path,
                '--log', self.log_path,
                '-r', 'pdf',
                '--report-dir', self.report_path,
                clf_file
            ]
            
            self.logger.info(f"执行TTman命令: {' '.join(cmd)}")
            
            # 执行命令
            result = subprocess.run(
                cmd,
                capture_output=True,
                text=True,
                timeout=3600  # 1小时超时
            )
            
            # 检查执行结果
            if result.returncode == 0:
                self.logger.info("测试执行成功")
                return {
                    "status": "completed",
                    "clf_file": clf_file,
                    "stdout": result.stdout,
                    "stderr": result.stderr
                }
            else:
                self.logger.error(f"测试执行失败: {result.stderr}")
                return {
                    "status": "failed",
                    "clf_file": clf_file,
                    "error": result.stderr
                }
                
        except subprocess.TimeoutExpired:
            self.logger.error("测试执行超时")
            return {
                "status": "timeout",
                "clf_file": clf_file,
                "error": "执行超时"
            }
        except Exception as e:
            self.logger.error(f"执行测试失败: {e}")
            return {
                "status": "error",
                "clf_file": clf_file,
                "error": str(e)
            }
    
    def get_report_file(self, clf_file: str) -> Optional[str]:
        """获取测试报告文件路径
        
        Args:
            clf_file: clf文件路径
            
        Returns:
            报告文件路径
        """
        try:
            # 从clf文件名提取测试用例名
            clf_name = os.path.basename(clf_file).replace('.clf', '')
            
            # 构建报告文件路径
            report_file = os.path.join(self.report_path, f"{clf_name}.pdf")
            
            if os.path.exists(report_file):
                return report_file
            return None
        except Exception as e:
            self.logger.error(f"获取报告文件失败: {e}")
            return None
    
    def get_log_file(self, clf_file: str) -> Optional[str]:

            clf_name = os.path.basename(clf_file).replace('.clf', '')
            
            # 构建日志文件路径
            log_file = os.path.join(self.log_path, f"{clf_name}.tlz")
            
            if os.path.exists(log_file):
                return log_file
            return None
        except Exception as e:
            self.logger.error(f"获取日志文件失败: {e}")
            return None

#### 4.4.5 任务执行器

```python
"""
任务执行器 - 统一调度CANoe、TSMaster和TTworkbench
"""
import threading
import time
import logging
from enum import Enum
from typing import Dict, Any, Optional
from .canoe_controller import CANoeController
from .tsmaster_controller import TSMasterController
from .ttworkbench_controller import TTworkbenchController

class TestToolType(Enum):
    CANOE = "canoe"
    TSMASTER = "tsmaster"
    TTWORKBENCH = "ttworkbench"

class TaskStatus(Enum):
    PENDING = "pending"
    RUNNING = "running"
    COMPLETED = "completed"
    FAILED = "failed"
    CANCELLED = "cancelled"

class TaskExecutor:
    """测试任务执行器"""
    
    def __init__(self, websocket_client):
        self.ws_client = websocket_client
        self.logger = logging.getLogger(__name__)
        self.current_task = None
        self.controller = None
        self._stop_event = threading.Event()
        self._task_thread = None
        
    def execute_task(self, task: Dict[str, Any]) -> bool:
        """执行测试任务"""
        if self.current_task is not None:
            self.logger.warning("已有任务正在执行")
            return False
        
        self.current_task = task
        self._stop_event.clear()
        
        # 启动任务执行线程
        self._task_thread = threading.Thread(target=self._execute, args=(task,))
        self._task_thread.start()
        return True
    
    def _execute(self, task: Dict[str, Any]):
        """任务执行主逻辑"""
        task_id = task.get("taskId")
        tool_type = task.get("toolType")
        config_path = task.get("configPath")
        test_items = task.get("testItems", [])
        
        try:
            # 更新状态为运行中
            self._update_status(task_id, TaskStatus.RUNNING)
            
            # 初始化控制器
            if tool_type == TestToolType.CANOE.value:
                self.controller = CANoeController()
            elif tool_type == TestToolType.TSMASTER.value:
                self.controller = TSMasterController()
            elif tool_type == TestToolType.TTWORKBENCH.value:
                self.controller = TTworkbenchController()
                # 配置TTworkbench路径
                self.controller.configure(
                    ttman_path=task.get("ttmanPath"),
                    workspace_path=task.get("workspacePath"),
                    log_path=task.get("logPath"),
                    report_path=task.get("reportPath")
                )
            else:
                raise Exception(f"不支持的测试工具类型: {tool_type}")
            
            # 连接测试软件
            if tool_type != TestToolType.TTWORKBENCH.value:
                if not self.controller.connect():
                    raise Exception("连接测试软件失败")
            
            # 加载配置
            if tool_type != TestToolType.TTWORKBENCH.value:
                if not self.controller.open_configuration(config_path):
                    raise Exception("加载配置文件失败")
            
            # 启动测量/仿真
            if tool_type == TestToolType.CANOE.value:
                self.controller.start_measurement()
            elif tool_type == TestToolType.TSMASTER.value:
                self.controller.start_simulation()
            # TTworkbench不需要启动仿真，直接执行clf文件
            
            # 执行测试项
            results = []
            for item in test_items:
                if self._stop_event.is_set():
                    self.logger.info("任务被取消")
                    break
                
                result = self._execute_test_item(item, tool_type)
                results.append(result)
                
                # 实时上报进度
                self._report_progress(task_id, item, result)
            
            # 停止测量/仿真
            if tool_type == TestToolType.CANOE.value:
                self.controller.stop_measurement()
            elif tool_type == TestToolType.TSMASTER.value:
                self.controller.stop_simulation()
            # TTworkbench执行完成后自动停止
            
            # 上报最终结果
            self._report_result(task_id, results)
            self._update_status(task_id, TaskStatus.COMPLETED)
            
        except Exception as e:
            self.logger.error(f"任务执行失败: {e}")
            self._update_status(task_id, TaskStatus.FAILED, str(e))
        finally:
            if self.controller:
                self.controller.disconnect()
            self.current_task = None
    
    def _execute_test_item(self, item: Dict[str, Any], tool_type: str) -> Dict[str, Any]:
        """执行单个测试项"""
        item_type = item.get("type")
        item_name = item.get("name")
        
        # TTworkbench执行clf文件
        if tool_type == TestToolType.TTWORKBENCH.value:
            clf_file = item.get("clfFile")
            if not clf_file:
                raise Exception("TTworkbench测试项缺少clf文件路径")
            
            result = self.controller.execute_test(clf_file)
            
            # 获取报告和日志文件
            report_file = self.controller.get_report_file(clf_file)
            log_file = self.controller.get_log_file(clf_file)
            
            return {
                "name": item_name,
                "type": item_type,
                "clf_file": clf_file,
                "result": result,
                "report_file": report_file,
                "log_file": log_file
            }
        
        # CANoe和TSMaster的测试项执行
        elif item_type == "signal_check":
            # 信号检查
            signal_name = item.get("signalName")
            expected_value = item.get("expectedValue")
            actual_value = self.controller.get_signal(signal_name)
            return {
                "name": item_name,
                "type": item_type,
                "expected": expected_value,
                "actual": actual_value,
                "passed": abs(actual_value - expected_value) < 0.01 if actual_value else False
            }
        
        elif item_type == "signal_set":
            # 信号设置
            signal_name = item.get("signalName")
            value = item.get("value")
            success = self.controller.set_signal(signal_name, value)
            return {
                "name": item_name,
                "type": item_type,
                "success": success
            }
        
        elif item_type == "test_module":
            # 执行测试模块
            if isinstance(self.controller, CANoeController):
                return self.controller.run_test_module(item_name)
            else:
                return self.controller.run_test_sequence(item_name)
        
        return {"name": item_name, "error": "未知的测试项类型"}
    
    def cancel_task(self) -> bool:
        """取消当前任务"""
        if self.current_task is None:
            return False
        self._stop_event.set()
        return True
    
    def _update_status(self, task_id: str, status: TaskStatus, message: str = None):
        """更新任务状态"""
        self.ws_client.send_message({
            "type": "TASK_STATUS",
            "taskId": task_id,
            "status": status.value,
            "message": message,
            "timestamp": int(time.time() * 1000)
        })
    
    def _report_progress(self, task_id: str, item: Dict[str, Any], result: Dict[str, Any]):
        """上报执行进度"""
        self.ws_client.send_message({
            "type": "LOG_STREAM",
            "taskId": task_id,
            "level": "INFO",
            "message": f"执行测试项: {item.get('name')}",
            "result": result,
            "timestamp": int(time.time() * 1000)
        })
    
    def _report_result(self, task_id: str, results: list):
        """上报测试结果"""
        self.ws_client.send_message({
            "type": "RESULT_REPORT",
            "taskId": task_id,
            "results": results,
            "timestamp": int(time.time() * 1000)
        })
```
